Clojure Language
----------------

Clojure is:
- a Lisp Dialect
- Java based
- dynamic
- functional (impure - haskell is a pure functional language)

Clojure offers:
- "lockless" concurrency - a feature which has garnered much attention
- macros, like ruby metaprogramming
